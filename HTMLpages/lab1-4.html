Common Html Tags

Block Elements

Block elements take up their own line and take up the full width of the page.

body tag: Defines the document's body Head tag: Defines information about the document html tag: Defines the root of an HTML document div tag: Defines a section in a document section tag: Defines a section in a document header tag: Defines a header for a document or section main tag: Specifies the main content of a document article tag: Defines an article nav tag: Defines navigation links footer tag: Defines a footer for a document or section p tag: Defines a paragraph aside tag: Defines content aside from the page content li tag: Defines a list item

Inline Elements

a tag: Defines a hyperlink strong tag: Defines important text em tag: Defines emphasized text img tag: Defines an image span tag: Defines a section in a document
